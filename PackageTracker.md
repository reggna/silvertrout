# Information #

The package tracker plugin keeps track of your packages and informs you of any updates that happen to them.

# Usage #

Commands:

  * !listpackages
  * !addpackage PACKETID
  * !removepackage PACKETID
  * !statuspackage PACKETID

# Supported services #

  * Swedish postal service
  * Schenker PrivPak
