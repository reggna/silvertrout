# Introduction #

Quizmaster is a plugin for silvertrout that can hold a quiz in a specific channel.

# Features #

There are many cool features in Quizmaster.

  * Toplist
  * Ranks
  * Trophies

# Instructions #

Commands:

  * !start
  * !stats [nick](nick.md)
  * !toplist
  * !help
  * !trophies
  * !suggest (Not implemented yet)
  * !report (Not implemented yet)
